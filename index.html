Buat Bot Telegram
Bot ini adalah asisten pribadi keuangan Anda di Telegram.
📱 Cara membuat Bot Telegram
  1. Buka Telegram → di kolom pencarian ketik @BotFather → tap
  2. Tap tombol Start atau ketik /start
  3. Ketik /newbot dan kirim
  4. BotFather akan minta nama bot — isi bebas, contoh: Tracker Keuangan Saya
  5. BotFather akan minta username bot — harus diakhiri bot, contoh: trackerku_bot
  6. Setelah berhasil, BotFather akan kirim token — copy seluruh angka dan huruf tersebut
Token tidak valid. Pastikan format: angka:huruf (minimal 35 karakter setelah titik dua)
Verifikasi Identitas
Chat ID adalah nomor unik akun Telegram Anda — digunakan agar bot tahu ke mana harus mengirim pesan.
📱 Cara mendapatkan Chat ID
  1. Buka Telegram → di kolom pencarian ketik @userinfobot → tap
  2. Tap tombol Start atau ketik /start dan kirim
  3. Bot akan langsung membalas pesan berisi data Anda seperti ini:
    Id: 1287197203
    First: Budi
    Last: Santoso
    Lang: id
  4. Copy angka di baris Id — itulah Chat ID Anda
  5. Paste di kolom di bawah ini
Chat ID tidak valid — harus berupa angka 5-15 digit

Aktivasi Berhasil!

Bot dan Google Sheet Anda sudah siap. Ikuti langkah berikut untuk mulai menggunakan:

1
Cek email Google Anda
Google Drive akan mengirim notifikasi bahwa sebuah sheet telah dibagikan ke Anda. Klik link tersebut untuk membuka Google Sheet tracker Anda.
2
Buka bot Telegram Anda
Buka Telegram → cari username bot yang baru Anda buat tadi (yang Anda daftarkan di step 2) → tap untuk membuka chat.
3
Ketik /start dan kirim
Bot akan membalas dengan pesan sambutan, link Google Sheet Anda, dan tombol menu navigasi.
4
Mulai catat transaksi!
Langsung ketik transaksi di bot, contoh: makan 25k atau gaji 3.5jt
⚠️ Penting: Bot tidak bisa mengirim pesan duluan sebelum Anda ketik /start. Ini adalah kebijakan Telegram untuk melindungi privasi pengguna.
