🤖 Bot Telegram
Bot ini adalah asisten pribadi keuangan kamu di Telegram.
📱 Cara membuat Bot Telegram
  1. Buka Telegram → cari @BotFather → tap
  2. Tap Start atau ketik /start
  3. Ketik /newbot dan kirim
  4. Isi nama bot (bebas), lalu username (harus diakhiri bot)
  5. Copy token yang diberikan BotFather
Format: angka:huruf — minimal 35 karakter setelah titik dua
Token tidak valid. Format: angka:huruf (min 35 karakter)
🆔 Chat ID Telegram
Chat ID adalah nomor unik akun Telegram kamu.
📱 Cara mendapatkan Chat ID
  1. Buka Telegram → cari @userinfobot → tap
  2. Tap Start atau ketik /start
  3. Bot akan balas dengan data kamu — copy angka di baris Id
Chat ID tidak valid — harus berupa angka 5-15 digit
